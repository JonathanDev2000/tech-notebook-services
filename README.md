## Sobre o Projeto

Site institucional da **TechNotebooks**, empresa especializada em manutenção e reparo de notebooks em Engenho de Dentro, Rio de Janeiro. O site foi desenvolvido do zero com HTML, CSS e JavaScript puro, focando em performance, SEO e experiência do usuário mobile-first.

**Site ao vivo:** [www.technotebooks.com.br](https://www.technotebooks.com.br)

---

## Funcionalidades

- Apresentação dos serviços oferecidos (diagnóstico, reparo, limpeza, upgrades)
- Formulário de solicitação de orçamento online
- Blog com artigos técnicos sobre manutenção de notebooks
- Design responsivo (mobile-first)
- SEO otimizado com sitemap.xml e meta tags
- Integração com WhatsApp para contato rápido
